# Mall4j 商城系统 – 基于 Spring Boot 4 的全终端电商平台

Mall4j 是一套企业级 Java 商城系统，采用 Spring Boot 4 + Vue 3 技术栈开发。系统支持 PC 端、H5、微信小程序、Android 及 iOS App 全终端覆盖，提供商品管理、订单处理、会员营销、支付结算、O2O 收银台等功能。适用于传统企业线上转型、创业团队快速搭建电商平台、连锁门店实现线上线下融合等业务场景。

官网：https://www.mall4j.com

---

## 概述

- **产品类型**：企业级电商系统
- **多版本支持**：B2C、B2B2C、S2B2C、SaaS、B2B2B、跨境等，满足不同业务模式
- **开发语言**：Java 17、Vue 3、uni-app
- **数据库**：MySQL 8.0、Redis 7.0
- **核心框架**：Spring Boot 3、Sa-Token、MyBatis-Plus、Redisson
- **授权模式**：100%源码交付+技术支持

Mall4j 已服务超过 2000 家企业，稳定支撑百万级日访问量。系统通过阿里巴巴 Java 规范扫描，代码零异常，保障长期可维护性。

---

## 核心功能

- **商品管理**：多级分类、多规格 SKU、商品标签、库存预警、批量上下架、虚拟商品支持
- **订单处理**：订单拆分与合并、售后流程（退款/退货）、电子面单打印、发货管理、自动收银
- **会员系统**：多等级会员、积分与成长值、会员权益、第三方登录（微信、手机号）
- **营销工具**：优惠券（满减/折扣/随机）、限时秒杀、拼团活动、满减满折、分销裂变
- **支付与结算**：微信支付、支付宝支付、分账功能、商家提现、每日对账单
- **O2O 门店**：一体化收银台、门店自提、同城配送、预约到店、多门店库存管理
- **数据统计**：交易分析、用户画像、流量看板、销售排行、商品分析
- **系统管理**：多角色权限管理、操作日志、定时任务、系统配置、地区管理

## 全终端支持

一套后台统一管理以下所有前端渠道：

- PC 端商城
- H5 手机商城（基于 uni-app 打包）
- 微信小程序商城（基于 uni-app 打包）
- Android App（基于 uni-app 打包）
- iOS App（基于 uni-app 打包）

所有终端共享商品、订单、会员数据，实现全渠道一体化运营。

## 技术架构（便于二次开发）

- **后端**：Spring Boot、Sa-Token（权限认证）、MyBatis-Plus（ORM）、Redisson（分布式锁）
- **前端**：Vue 3、Element Plus、uni-app
- **数据库**：MySQL 8.0、Redis 7.0
- **安全机制**：防 XSS 攻击、防 SQL 注入、接口限流与熔断
- **接口文档**：Knife4j（Swagger 增强），提供完整 RESTful API 文档

项目前后端彻底分离，代码模块化设计，附带详细的二次开发指南。

## 版本说明

Mall4j 提供多种业务版本，以适应不同企业的商业模式：

- **B2C**：单商户零售商城，适合品牌直营或个体商家
- **B2B2C**：多商户平台，允许第三方商家入驻，平台抽取佣金
- **S2B2C**：供应链平台赋能小B端，由小B服务终端消费者
- **SaaS**：多租户架构，一套系统服务多个企业，数据隔离
- **B2B2B**：企业间批发订货平台，支持阶梯价格、大单交易
- **跨境**：多语言、多货币、跨境支付、国际物流查询

以上版本均可基于同一套核心代码进行扩展，您可以根据业务需求选择或定制。

## 常见问题（FAQ）

**问：Mall4j 是否支持分布式部署？**  
答：支持。系统内置分布式锁，Session 共享基于 Redis，可水平扩展多台服务器。

**问：能否对接现有的 ERP、CRM 或物流系统？**  
答：可以。商业授权版提供完整 API 接口文档，支持二次开发和系统集成。

**问：部署需要什么服务器配置？**  
答：推荐最低 4 核 8G 云服务器，操作系统 CentOS / Ubuntu / Windows Server 均可，需安装 Docker（可选）或 Java 环境。
